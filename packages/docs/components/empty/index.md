## Empty API

### Empty 属性

| 属性               | 说明             | 类型      | 默认值 |
| ------------------ | ---------------- | --------- | ------ |
| `show-description` | 是否展示描述信息 | `boolean` | `true` |

### Empty 插槽

| 插槽名     | 说明           |
| ---------- | -------------- |
| `content`  | 自定义内容文字 |
| `iconName` | 自定义图标     |

## Radio API

### Radio 属性

| 属性                      | 说明           | 类型                            | 默认值 |
| ------------------------- | -------------- | ------------------------------- | ------ |
| `v-model` / `model-value` | 选中项绑定值   | `string` / `number` / `boolean` | -      |
| `disabled`                | 是否禁用单选框 | `boolean`                       | false  |

# Throttle 节流

节流指令用于控制事件的触发频率，在指定时间内只执行一次事件处理函数。

## API

### 参数

| 参数名    | 说明             | 类型    | 默认值 |
| --------- | ---------------- | ------- | ------ |
| delay     | 延迟时间（毫秒） | number  | 300    |
| immediate | 是否立即执行     | boolean | false  |

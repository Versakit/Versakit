# Avatar 头像

Avatar 组件可以用来代表人物或对象， 支持使用图片、图标或者文字作为 Avatar。

## 基础用法

<p>
使用 <code>shape</code> 和 <code>size</code> 属性来设置 Avatar 的形状和大小。
</p>

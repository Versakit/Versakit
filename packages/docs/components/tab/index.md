# Tabs 标签页

分隔内容上有关联但属于不同类别的数据集合。

## 基础用法

Tabs 组件提供了选项卡功能，通过 `tabList` 绑定当前激活的 `Tab` 名称。

<demo vue="./example/base.vue"></demo>

## Tabs API

### Tabs 属性

| 属性      | 说明              | 类型    | 默认值 |
| --------- | ----------------- | ------- | ------ |
| `tabList` | Tabs 中绑定的数据 | `array` | -      |

### Tabs 插槽

| 名称   | 说明              |
| ------ | ----------------- |
| 自定义 | 为 Tab 的唯一名称 |

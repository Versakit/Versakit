# LongPress 长按指令

长按指令用于检测元素的长按事件。支持自定义触发时长，同时支持移动端触摸事件和桌面端鼠标事件。

## API

### 指令值

| 参数     | 说明                               | 类型         | 默认值 |
| -------- | ---------------------------------- | ------------ | ------ |
| duration | 触发长按事件需要按住的时间（毫秒） | `number`     | `500`  |
| callback | 长按触发的回调函数                 | `() => void` | -      |

### 示例类型定义

```ts
interface LongPressOptions {
	duration?: number
	callback?: () => void
}
```

### 注意事项

1. 同时支持移动端触摸事件和桌面端鼠标事件
2. 在触发长按事件之前松开或离开元素会取消事件
3. 默认会阻止长按时的默认行为（如选中文本）
